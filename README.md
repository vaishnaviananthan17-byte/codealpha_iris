# Iris Flower Classification 

This project is a Machine Learning classification project that predicts the species of an Iris flower using flower measurements.

# Project Overview

The Iris dataset contains flower measurements such as:

- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

Using these features, the Machine Learning model predicts the flower species:

- Iris-setosa
- Iris-versicolor
- Iris-virginica
